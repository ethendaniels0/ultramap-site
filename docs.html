Documentation

Welcome to ultramap documentation. Learn how to get started and make the most of your whiteboarding experience.

Getting Started

ultramap is designed to be simple and intuitive. After purchasing, you'll receive a download link for your platform.

Installation

Download ultramap for your operating system and follow the installation instructions. ultramap is available for macOS, Windows, and Linux.

First Steps

When you first open ultramap, you'll be greeted with a blank canvas. Here are the basics:

  • Click and drag to pan around your canvas
  • Use the toolbar to add notes, shapes, and frames
  • Double-click anywhere to create a new note
  • Press / to access the AI assistant

Features

AI-Powered Brainstorming

ultramap's AI assistant helps you brainstorm and organize ideas. Simply press / anywhere on your canvas to activate the AI assistant.

The AI can help you:

  • Generate ideas and suggestions
  • Organize your thoughts into structured frameworks
  • Create outlines and plans
  • Expand on existing concepts

Whiteboard Tools

ultramap includes all the essential whiteboarding tools you need:

  • Notes: Add text notes anywhere on your canvas
  • Sticky Notes: Create colored sticky notes for quick ideas
  • Frames: Group related content together in frames
  • Connectors: Draw lines and arrows to show relationships
  • Shapes: Add rectangles, circles, and other shapes
  • Images: Embed images directly on your canvas

Embedded Content

Embed external content directly into your canvas:

  • Web pages and links
  • Videos and media
  • Documents and PDFs
  • Code snippets

Keyboard Shortcuts

Speed up your workflow with these keyboard shortcuts:

  • N - Create new note
  • F - Create frame
  • / - Open AI assistant
  • Space + Drag - Pan canvas
  • Cmd/Ctrl + Z - Undo
  • Cmd/Ctrl + Shift + Z - Redo
  • Cmd/Ctrl + S - Save
  • Cmd/Ctrl + D - Duplicate selection

Tips and Tricks

For Writers

  • Use frames to organize chapters or story arcs
  • Create character maps with sticky notes and connectors
  • Use the AI to help develop plot points and character backgrounds

For Designers

  • Create mood boards with embedded images
  • Organize design systems with frames
  • Use the AI to generate design concepts and alternatives

For Engineers

  • Diagram system architecture with shapes and connectors
  • Embed code snippets for reference
  • Use frames to represent different services or components

For Students

  • Create visual study guides and mind maps
  • Connect concepts with arrows and lines
  • Use the AI to help explain complex topics

Troubleshooting

Performance Issues

If ultramap is running slowly:

  • Close unused boards to free up memory
  • Reduce the number of embedded images on large canvases
  • Clear your cache from the settings menu

Syncing Issues

If your boards aren't syncing properly:

  • Check your internet connection
  • Force a sync from the menu bar
  • Restart ultramap
